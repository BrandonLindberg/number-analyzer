# Rust Number Analyzer

This is a simple Rust program that takes numbers as inputs, writes them to a file, retrieves and reads that file, and then analyzes the sign of the number.

## Instructions for Build and Use

Steps to build and/or run the software:

1. In the root project folder, run the command 'cargo run'

Instructions for using the software:

1. Input as many numbers as you desire into the prompt.
2. Type 'done'.
3. Numbers will be displayed in terminal.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Install rust-tools
* Install rust-analyzer extension if using VSCode
